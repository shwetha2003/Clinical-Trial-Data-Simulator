# Clinical-Trial-Data-Simulator

A comprehensive platform for simulating clinical trial data, designing trials, and analyzing results.

## Features

- **Patient Simulation**: Generate realistic patient demographics and lab results
- **Trial Design**: Design clinical trials with statistical power calculations
- **Data Validation**: Comprehensive data quality checks and validation
- **Statistical Analysis**: ANOVA, t-tests, correlation analysis
- **Web Dashboard**: Real-time visualization of trial data
- **Export Capabilities**: Export data to CSV and JSON formats
